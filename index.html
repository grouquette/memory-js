Mon Projet Memory

Bienvenue sur mon projet de Memory!
C'est un jeu qui fait travailler la mémoire, dans le but de retrouver des paires de cartes identiques.

Règles du jeu

On clique sur 2 cartes faces cachées. les deux cartes se retournent.
Si les images sont identiques, les deux cartes disparaissent.
Dans le cas contraire les cartes se retournent face cachées et on doit recommencer.
La partie est terminée quand toutes les cartes ont étéassemblées par paires. Le score dépend du nombre d'erreur faites avant de toutes les retourner.

Si vous voulez vous souvenir de votre score, Inscrivez-vous via le formulaire ou connectez-vous.
Si vous voulez relancer une partie, pressez la barre d'espace.
