Mon Projet Memory

Bienvenue sur mon projet de Memory. c'est un jeu qui fait travailler la mémoire, dans le but de retrouver des paires de cartes identiques.

Règles du jeu

On clique sur 2 cartes face cachées. les deux cartes se retournent. Si les images sont identiques, le score augmente. Dans le cas contraire les cartes se retournent face cachées et on doit recommencer. Le score dépend du nombre d'erreur faites avant de toutes les retourner.
